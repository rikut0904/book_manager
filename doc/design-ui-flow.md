# 画面設計・画面フロー

## 画面一覧
- 認証: ログイン/新規登録/パスワード再設定
- 所蔵一覧: 検索/シリーズ絞り込み
- 書籍登録: 手動/ISBN/バーコード
- 書籍詳細: 所蔵メモ/シリーズ情報/お気に入り/おすすめ
- お気に入り一覧: 単巻/シリーズ
- 次に買う本: シリーズ次巻 + 手入力
- おすすめ一覧: 全体公開
- おすすめ投稿
- ユーザー検索
- プロフィール: 公開範囲設定、統計表示、フォロー
- 設定: シリーズ管理、プロフィール公開設定

## 基本フロー
1) ログイン
2) 所蔵一覧 → 書籍登録
3) 書籍詳細でシリーズ自動判定を確認 → 必要なら手動修正
4) お気に入り登録（単巻/シリーズ）
5) 次に買う本はシリーズ次巻 + 手入力で管理
6) おすすめ投稿 → 全体公開
7) ユーザー検索 → プロフィール閲覧/フォロー

## 権限
- 書誌マスタ更新: ユーザーからは不可
- プロフィール: 公開/フォロワー限定切替
- おすすめ: 全体公開
